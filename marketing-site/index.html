Cabalmail

Self-hosted email on your own AWS account.

Placeholder home page. Design and copy for this site are scheduled for a later milestone. For now this page exists so the project has a public marketing surface that hosts the legal pages required for SMS verification.

Cabalmail provides three mail tiers (IMAP, inbound SMTP, outbound SMTP with DKIM signing) backed by Cognito authentication, EFS mailstores, and a self-hosted React admin UI. Per-address provisioning lets you spin up disposable addresses per vendor or per use, and revoke them when they get burned.

The full source code is open source under AGPLv3.
